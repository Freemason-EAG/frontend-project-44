## Installation
```bash
git clone https://github.com/Freemason-EAG/frontend-project-44.git
cd frontend-project-44
make install
npm link
```

## How to Play
To play, use the following commands:
- brain-even
- brain-calc
- brain-gcd
- brain-progression
- brain-prime
